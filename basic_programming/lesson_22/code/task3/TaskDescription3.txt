написать приложение для учета и проведения операций с различными медиа-материалами:
книги, журналы и тд

Базавая функциональность:
- добавить новый носитель
- отобразить информацию о доступных элементах (материалах)

==================================================================

Класс - MediaItem - общий класс для всех медиа-материалов с базовыми свойствами и методами

- id
- название
- автор
- год издания

Класс Book :
- жанр
- количество страниц

Класс Magazine:

- номер выпуска


Класс LibraryCatalog - обеспечивал бы хранение массива MediaItems и методов доступа к ним
