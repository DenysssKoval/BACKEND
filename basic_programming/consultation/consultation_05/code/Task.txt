Создать класс Car
у автомобиля должны быть следующие параметры:
- количество колес
- диаметр колес
- объем двигателя (см3)
- тип двигателя (бензин / дизель/ электрик)
- марка
- модель

Машина должна уметь:
 - заводить/выключать двигатель (печатать сообщение об этом)
 - подавать сигнал

Создать массив из трех авто и напечатать инфо о них
